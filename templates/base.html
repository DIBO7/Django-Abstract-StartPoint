Django-Abstract-StartPoint is an ideal startpoint for your new django project

    It comes wimple and editable 'base.html' and 'index.html'

  • 2) password change done, password change form, password reset complete, password reset confirm, password reset done, password reset form

    It can be easily integrated to django projects
